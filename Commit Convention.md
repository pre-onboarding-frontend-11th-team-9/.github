# 커밋 컨벤션

- Gitmoji 사용 안함 (고르는데 시간 걸림. 비효율적)
- **Conventional Commit**을 준수하여 작성
- 소문자, 동사원형 사용, 마침표 지양
- 예시 타입 : feat, fix, refactor, docs, style(코드 포맷 변경), design(UI 변경), test, ci, perf, chore, rename, move, remove
- 커밋 예시 : `feat: 회원가입 기능 구현`
